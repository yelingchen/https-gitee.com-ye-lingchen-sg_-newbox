# 下拉选择框操作

## selectOptions(selector, list, value, key_param, val_param)

select渲染option

### 语法
```javascript
selectOptions(selector, list, value, key_param, val_param)
```

### 参数
| 参数名     | 类型   | 必填 | 说明         |
| ---------- | ------ | ---- | ------------ |
| selector   | string | 是   | 选择器       |
| list       | Array  | 是   | 列表         |
| value      | any    | 是   | 选中值       |
| key_param  | string | 是   | 键参数       |
| val_param  | string | 是   | 值参数       |

### 示例
```javascript
// 渲染省份下拉框
var provinces = [
    {id: "1", name: "北京市"},
    {id: "2", name: "上海市"},
    {id: "3", name: "广东省"}
];

selectOptions("#province", provinces, "2", "id", "name");
```

## getSelectData(selector)

获取select数据

### 语法
```javascript
getSelectData(selector)
```

### 参数
| 参数名   | 类型   | 必填 | 说明   |
| -------- | ------ | ---- | ------ |
| selector | string | 是   | 选择器 |

### 返回值
返回select控件的数据

### 示例
```javascript
// 获取id为province的下拉框数据
var provinceData = getSelectData("#province");
console.log(provinceData);
```

---

## getSelectText(selector)

获取select文字

### 语法
```javascript
getSelectText(selector)
```

### 参数
| 参数名   | 类型   | 必填 | 说明   |
| -------- | ------ | ---- | ------ |
| selector | string | 是   | 选择器 |

### 返回值
返回select控件的显示文字

### 示例
```javascript
// 获取id为province的下拉框选中项的文本
var provinceText = getSelectText("#province");
console.log("选中的省份: " + provinceText);
```
