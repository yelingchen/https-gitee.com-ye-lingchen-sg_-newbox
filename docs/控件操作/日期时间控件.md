# 日期时间控件操作

## getDateVal(selector, fmt)

日期时间-取值

### 语法
```javascript
getDateVal(selector, fmt)
```

### 参数
| 参数名   | 类型   | 必填 | 说明   |
| -------- | ------ | ---- | ------ |
| selector | string | 是   | 选择器 |
| fmt      | string | 是   | 格式   |

### 返回值
返回日期时间值

### 示例
```javascript
// 获取id为birthDate的日期控件值，格式为YYYY-MM-DD
var birthDate = getDateVal("#birthDate", "YYYY-MM-DD");
console.log("出生日期: " + birthDate);
```

## formatDate(date, fmt)

日期格式转换

### 语法
```javascript
formatDate(date, fmt)
```

### 参数
| 参数名 | 类型   | 必填 | 说明     |
| ------ | ------ | ---- | -------- |
| date   | Date   | 是   | 日期对象 |
| fmt    | string | 是   | 格式     |

### 返回值
返回格式化后的日期字符串

### 示例
```javascript
// 格式化当前日期
var now = new Date();
var formattedDate = formatDate(now, "YYYY-MM-DD HH:mm:ss");
console.log("当前时间: " + formattedDate);
```
