# 树状菜单操作

## renderTreeMenu(id, datas, params)

渲染树状菜单

### 语法
```javascript
renderTreeMenu(id, datas, params)
```

### 参数
| 参数名 | 类型   | 必填 | 说明             |
| ------ | ------ | ---- | ---------------- |
| id     | string | 是   | 控件id           |
| datas  | Array  | 是   | 树状菜单的数据   |
| params | Object | 是   | 配置参数         |

### 示例
```javascript
// 渲染树状菜单
var treeData = [
    {
        id: "1",
        name: "根节点",
        children: [
            {
                id: "2",
                name: "子节点1"
            },
            {
                id: "3",
                name: "子节点2"
            }
        ]
    }
];

var treeParams = {
    expandAll: true
};

renderTreeMenu("treeMenu", treeData, treeParams);
```

## get_tree_data(id)

获取树状菜单渲染的数据

### 语法
```javascript
get_tree_data(id)
```

### 参数
| 参数名 | 类型   | 必填 | 说明   |
| ------ | ------ | ---- | ------ |
| id     | string | 是   | 控件id |

### 返回值
返回指定控件id对应的已渲染树状菜单的数据

### 示例
```javascript
// 获取树状菜单数据
var treeData = get_tree_data("treeMenu");
console.log("树状菜单数据:", treeData);
```
