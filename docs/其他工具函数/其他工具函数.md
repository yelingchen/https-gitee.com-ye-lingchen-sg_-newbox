# 其他工具函数

## getHtmlById(id)

通过id获取html

### 语法
```javascript
getHtmlById(id)
```

### 参数
| 参数名 | 类型   | 必填 | 说明   |
| ------ | ------ | ---- | ------ |
| id     | string | 是   | 节点id |

### 返回值
返回节点的html

### 示例
```javascript
// 获取指定元素的HTML内容
var htmlContent = getHtmlById("contentDiv");
console.log(htmlContent);
```

---

## getValueById(id)

通过ID获取控件value值

### 语法
```javascript
getValueById(id)
```

### 参数
| 参数名 | 类型   | 必填 | 说明   |
| ------ | ------ | ---- | ------ |
| id     | string | 是   | 节点id |

### 返回值
返回指定元素的value值

### 示例
```javascript
// 获取id为username的输入框的值
var username = getValueById("username");
console.log(username);
```

---

## setValueById(id, value, text)

通过ID设置控件value值

### 语法
```javascript
setValueById(id, value, text)
```

### 参数
| 参数名 | 类型   | 必填 | 说明         |
| ------ | ------ | ---- | ------------ |
| id     | string | 是   | 节点id       |
| value  | any    | 是   | 值           |
| text   | string | 否   | 可选文本     |

### 示例
```javascript
// 设置id为username的输入框的值为"张三"
setValueById("username", "张三");

// 设置id为email的输入框的值和文本
setValueById("email", "zhangsan@example.com", "张三的邮箱");
```
